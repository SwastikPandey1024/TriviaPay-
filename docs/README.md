# 💡 TriviaPay – Campus FinTech on Blockchain

A blockchain-powered student finance ecosystem built using Algorand.

## 🚀 Features
- Split bills automatically
- Campus wallet payments
- Goal-based savings tracker

## ⚙️ Tech Stack
- React.js
- Algorand (AlgoSDK)
- Node.js

## 🧠 Architecture
Frontend → React  
Backend → Node.js  
Blockchain → Algorand  

## ▶️ Run Project

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
cd backend  
npm install  
node server.js
